# Project Overview

## Project Name

VICCS Antigravity: O Cérebro Global (AI Knowledge Base)

## Description

A context operating system designed for AI Agents (Gemini, Claude, GPT) to transform generic LLMs into Elite Engineers. It contains "muscular memory", architecture patterns, and strategic vision for VICCS Antigravity.

## Tech Stack

- **Languages**: Markdown (Documentation), JSON (Configuration)
- **Frameworks**: None (Pure Knowledge Base)
- **Tools**: AI Agents (Gemini, Claude, GPT), File System
- **Services**: None

- **Skills Library**: A vast collection of **2622 technical skills** for AI agents, covering:

  - Backend: Node.js, Python, Go, Rust, Java, .NET, FastAPI, Django
  - DevOps: Docker, Kubernetes, Terraform, CI/CD, AWS, Azure, GCP
  - Security: Penetration Testing, Threat Modeling, OWASP, Red Team Tactics
  - AI/ML: LangChain, RAG, Agent Orchestration, Prompt Engineering
  - Game Dev: Unity, Unreal, Godot, FiveM (QBCore/VRP), Hytale Modding
  - Marketing: SEO, Content, Analytics, CRO, Email Sequences
  - And many more specialized domains...
- **Game Design Documents (GDD)**: Strategic and creative documentation.
- **Workflows**: Step-by-step procedures for complex tasks.
- **Global Context Router**: A master index for skill navigation.
- **Project Rules**: Skill creation guidelines and routing rules.
